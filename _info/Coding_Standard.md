# Coding Standard
* All code is written in english
* File names are written in PascalCase
* Class, observable collection, and method names are written in PascalCase
* WPF element names are written in PascalCase
* Variables are written in camelCase

### Standards of Comments:
* All comments in start with a space after the initializing symbols
* Explanatory comments are written above relevant code 
* Explanatory comments have one blank line above unless the line above starting or ending bracket
* Clarifying comments are written on the same line as the code its clarifying
