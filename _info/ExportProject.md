# How to export the Project

1. Right click the solution => Hover over "Add" => Click "New Project"
2. Choose the "Setup Project" template
3. Name it what you want but to make these instuctions easier to understand it's gonna be refered to as "CafeBullen"
3. Double click the newly created project CafeBullen and left click on "User's Desktop" on the left side of the window
4. Right click on the right side where Name and Type are columns and hover over "Add" => Click "Project Output"
5. Choose the "Publish Items" option
6. Build the setup project
7. Open the file explorer and navigate to the setup project folder, in this case "CafeBullen" => Debug and "CafeBullen"
8. Navigate to the newly created "TE4POS.exe" and double click on it
9. You have now successfully exported the project.
